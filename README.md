# README #

Readme for Multi-Sudoku-solver

### Instructions ###

* Input file is "puzzles.txt"
* Output file is "solutions.txt"

### Format ###

* 9 x 9 grid
* No spaces between digits
* Use "0" for blanks
* Use a blank line between puzzles

### Example ###

    306508400
    520000000
    087000031
    003010080
    900863005
    050090600
    130000250
    000000074
    005206300
    
    800000000
    003600000
    070090200
    050007000
    000045700
    000100030
    001000068
    008500010
    090000400

### Notes ###

* Solutions will be in the same order as input
* Requires Apache Commons Lang library
